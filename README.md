# outreach-sent-message-scraper

Tampermonkey userscript for exporting sent message content from Outreach sequence message links.

## Included

- `src/step-1-collect-message-links.user.js` (Step 1: collect/export message links CSV)
- `src/step-2-export-message-bodies.user.js` (Step 2: load links CSV and export bodies)

## Notes

- This repo intentionally ignores local CSV exports and local debug files.
